# 당근마켓 클론 코딩

멋쟁이사자처럼 26기 2주차 2조 과제 — 당근마켓 메인 페이지 HTML/CSS 클론

---

## 프로젝트 소개

당근마켓(daangn.com) 메인 페이지를 HTML과 CSS만으로 구현한 클론 코딩 프로젝트입니다.
헤더, 메인 콘텐츠(서비스 소개 섹션), 푸터로 구성되어 있으며, 각 섹션을 별도 파일로 나누어 개발 후 `index.html`에 통합했습니다.

---

## 주요 구현 내용

| 섹션       | 설명                                                                       |
| ---------- | -------------------------------------------------------------------------- |
| **헤더**   | 당근 로고(SVG), 네비게이션 메뉴, 검색창, 채팅하기 버튼                     |
| **히어로** | 메인 타이틀, 설명 문구, Google Play / App Store 다운로드 버튼, 메인 이미지 |
| **본문**   | 중고거래, 동네생활, 알바, 동네업체 서비스 소개 섹션                        |
| **푸터**   | 앱 다운로드 버튼, 사이트 메뉴, 회사 정보, SNS 링크, 이용약관               |

---

## 파일 구조

```
likelion_week2_2/
├── index.html        # 전체 통합 페이지
├── index.css         # 전체 통합 스타일
├── carrot.html       # 헤더 + 히어로 섹션
├── carrot.css        # 헤더 + 히어로 스타일
├── body.html         # 본문 서비스 소개 섹션
├── body.css          # 본문 스타일
├── footer.html       # 푸터 섹션
├── footer.css        # 푸터 스타일
└── images/           # 이미지 리소스
```
